# data_engineering_python_task
Data Engineering Community LaunchPad Python task

The tasks are designed to demonstrate the use of Python for Data Engineering

## Clone Task Repository
Run the following:
```
git clone https://github.com/kabiromohd/data_engineering_python_task.git

cd data_engineering_python_task
```

## Requirements and Dependencies
To run the codes in the different tasks the dependancies have to be installed
```
pip install -r requirements.txt
```

## Populate credentials
Make a copy of ```.env_test``` and rename it ```.env```, then update the relevant credentials as appropriate
